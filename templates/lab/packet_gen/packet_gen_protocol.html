• Using the Packet Generator

    1. Use the Experiment drop-down menu to select the experiment you will be generating packet labels or models for.
    2. Select one of the three provided options:
      • Generate labels for existing seed packets: Returns a .CSV containing a brief summary of information contained in the select Experiment's StockPackets.
      • Preview generated stock packets for next year: Returns a .CSV with a preview of what Experiment's StockPacket table will look like once the generator script has run. The script will advance generations and pedigrees according to information stored in the Experiment's ObsPlots.
      • Push generated stock packets from preview onto DB: After you've looked over the generator preview, select this object to actually create those StockPacket objects in the database.
    3. Click on the confirmation checkbox and click submit.
