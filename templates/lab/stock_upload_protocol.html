Template Guide:

Upload Field Corresponding model Required Details
Used Stock False Enter 1 if the Stock was used in an experiment and leave blank otherwise.
Collected Stock False Enter 1 if the Stock was collected from an experiment and leave blank otherwise.
Experiment Name ObsTracker False The Experiment that this Stock will be associated with. Can be left blank.
Seed ID Stock True A unique identifier for this Stock.
Seed Name Stock False A human readable or alternative name for this Stock.
Cross Type Stock False The breeding method that was used to yield this particular Stock. Cross types should be entered as follows (excluding the text in parantheses): SF (selfing), SB (sibbing), CR (paired row crossing), TC (topcrossing), BC (backcrossing), FC (free-crossing), IN (intermating).
Pedigree Stock True The Stock's Pedigree.
Stock Status Stock False A descriptor of the current role of the Stock in an experiment's progress.
Stock Date Stock False Date the Stock was inventoried.
Inoculated Stock False Whether or not the Stock was inoculated. 1 for yes, leave blank for no.
Stock Comments Stock False Comments.
Binomial / Species / Population Passport False Information pertaining to the Stock's genotypic information.
Plot ID / Field Name ObsTracker False The ID's of any ObsPlots or Fields associated with the Stock.
Username / Collection Date / Method / Collection Comments Collecting False Information for the related Collecting tag.
Organization / First Name / Last Name / Phone / Email / Source Comments People False Information for the related Source tag.
