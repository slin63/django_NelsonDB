Username IsolateStock -> Collection The username of the person who curated this IsolateStock. (e.g. Enter {{user.username}} if you're the one who brought it into this world.)
Stock Date / Method / Collection Comments IsolateStock -> Collection When the IsolateStock was collected (or catalogued if not possible), what method was used to create it, and any relevant comments.
Binomial / Alias / Population / Race / Subtaxa IsolateStock -> Taxonomy Taxonomical information for the IsolateStock
Organization / Names / Phone / Email / Source Comments IsolateStock -> People Contact information for whoever provided the IsolateStock if it was not an in-house concoction.
Source IDs IsolateStock -> ObsTracker Fill these columns with the IDs of other objects in the database if the IsolateStock was created using them or their materials.
