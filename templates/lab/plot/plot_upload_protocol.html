Template Guide:

Upload Field Corresponding model Details
Plot ID ObsPlot Unique identifier for the ObsPlot object, formatted Experiment + 4 digit code (e.g. 14FL0001, 16GGU0015).
Experiment Name Experiment A 4-5 character formatted name (e.g. 14FL, 16GGU) referring to an existing experiment. If the experiment does not exist, one will be created with this name.
Source Seed ID Stock Unique identifier for the Stock object. If the Stock object does not exist, one will be created with this name.
Seed Name Stock Generic identifier for the Stock object. If the Stock object does not exist, one will be created with this name.
Population Stock -> Taxonomy Population with which the Stock is associated.
Pedigree Stock Pedigree with which the Stock is associated.
Field Name Field Field the plot is planted in. If the Field object does not exist, one will be created with this name.
Planting Year Field Year the plot was planted in.
Generation ObsPlot The generation of this ObsPlot's source seed (does not define gen of the Stock.)
Pollination type ObsPlot The pollination type of the plot. Must be entered as displayed in this list, not including text in parentheses: SF (selfing), SB (sibbing), CR (crossing).
Shell Type ObsPlot How the plot is being shelled. Either 'single', 'multi', or 'bulk'.
Row/Range ObsPlot Row / range coordinates of the plot.
Plot ObsPlot Third identifier for the plot.
Block ObsPlot Plot's block.
Rep ObsPlot Plot's rep.
Kernel Num ObsPlot Number of kernels planted for the plot.
Planting Date Experiment Date plot was planted.
Harvest Date ObsPlot Harvest date for the plot.
Organization Stock -> People The organization that provided the Stock.
First Name Stock -> People The first name of the person that provided the Stock.
Last Name Stock -> People The last name of the person that provided the Stock.
