Template Guide:

Upload Field Corresponding model Required Details
Plot ID ObsPlot True Unique identifier for the ObsPlot object, formatted Experiment + 4 digit code (e.g. 14FL0001, 16GGU0015).
Experiment Name Experiment True A 4-5 character formatted name (e.g. 14FL, 16GGU) referring to an existing experiment. If the experiment does not exist, one will be created with this name.
Source Seed ID Stock True Unique identifier for the Stock object. If the Stock object does not exist, one will be created with this name.
Seed Name Stock False Generic identifier for the Stock object. If the Stock object does not exist, one will be created with this name.
Is Male ObsPlot False Fill with 1 (a male) or leave blank (female). Only needs to be filled out for FreeCrossing and BackCrossing.
Cross Target ObsPlot False The Plot ID of the plot with which this plot is being crossed with. For FreeCrossing, females should specify their male source plants and male plants should leave this field blank. BackCrossing plots should also only specify targets if the plot is female. Again, male plots can be left blank.
Population Stock -> Taxonomy False Population with which the Stock is associated.
Pedigree Stock False Pedigree with which the Stock is associated.
Field Name Field True Field the plot is planted in. If the Field object does not exist, one will be created with this name.
Planting Year Field True Year the plot was planted in.
Generation ObsPlot False The generation of this ObsPlot's source seed (does not define gen of the Stock.) Inbred to indicate an inbred.
Pollination type ObsPlot False The pollination type of the plot. Must be entered as displayed in this list, not including text in parentheses: SF (selfing), SB (sibbing), CR (paired row crossing), TC (topcrossing), BC (backcrossing), FC (free-crossing), IN (intermating).
Shell Type ObsPlot False How the plot is being shelled. Either 'single' or 'bulk'.
Row/Range ObsPlot False Row / range coordinates of the plot.
Plot ObsPlot False Third identifier for the plot.
Block ObsPlot False Plot's block.
Rep ObsPlot False Plot's rep.
Kernel Num ObsPlot False Number of kernels planted for the plot.
Planting Date Experiment True Date plot was planted.
Harvest Date ObsPlot False Harvest date for the plot.
Organization Stock -> People False The organization that provided the Stock.
First Name Stock -> People False The first name of the person that provided the Stock.
Last Name Stock -> People False The last name of the person that provided the Stock.
