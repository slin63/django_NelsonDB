Template Guide:

Upload Field Corresponding model Required Details
Observation Unit Measurement True The unique ID of the model that this measurement was made for. E.g. a plot's plot_id, a stock's seed_id, or an isolate's isolate_id.
Experiment Measurement True The experiment that this measurement will be attached to. This experiment does not have to be the same as the experiment with which the Observation Unit is associated.
Parameter Name Measurement True The case-sensitive name of the parameter name that this measurement was taken for. Parameters can be viewed at the Measurement Parameter table.
Username Measurement False Username of the user who collected the data. E.g. {{ user.username }} if you were the one who collected the data.
DateTime Measurement False The time at which the data was taken.
Value Measurement True The value of the Measurement.
Measurement Comments Measurement False Any additional comments related to the Measurement.
